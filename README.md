# 图书借阅管理系统
基于 SpringBoot3 + MyBatis-Plus + MySQL + Redis 实现的校园图书借阅管理系统，包含用户登录认证、图书查询、借阅、归还、续借、借阅权限、逾期管理、接口限流、缓存优化等完整业务功能。

## 技术栈
- 核心框架：SpringBoot、Spring MVC
- ORM 框架：MyBatis-Plus
- 数据库：MySQL
- 连接池：HikariCP
- 缓存：Redis Redisson（限流/分布式控制）
- 登录认证：JWT
- 工具：Lombok、Hutool、Slf4j
- 构建工具：Maven

## 核心功能
- 用户登录/注册（JWT 身份认证）
- 图书列表、查询、详情展示
- 图书借阅、归还、续借业务
- 借阅限制（最大借阅数、最长天数）
- 逾期判断与状态管理
- 接口限流（防刷/保护系统）
- Redis 缓存热点数据 + 空值缓存（防穿透）
- 逻辑删除、统一响应结果、全局异常处理

## 项目亮点
- 负责核心业务模块开发，基于 Spring Boot + MyBatis-Plus 实现图书查询、借阅、续借、归还、用户认证授权等 RESTful 接口，搭建全局异常处理器与统一接口响应；
- 实现 JWT 双令牌认证机制，结合 Redis 维护 Token 黑名单，自定义权限注解完成多角色细粒度权限管控，保障接口访问安全无越权；
- 通过 Redisson 分布式锁防止图书借阅超卖，使用 MyBatis-Plus 乐观锁处理库存与借阅记录的并发更新冲突，解决高并发下的数据一致性问题；
- 优化系统查询性能，采用 Redis 缓存热门图书，空值缓存规避缓存穿透，核心查询接口响应时间从 200ms 降至 60ms，缓存命中率达 70%；
- 加入接口限流机制，防止恶意请求与突发流量冲击，提升系统稳定性与防护能力；
- 规范分层架构、统一返回封装、全局异常处理、逻辑删除、自动填充等；

## 快速启动
1. 创建数据库 library_db
2. 执行 sql/library_db.sql
3. 修改 application.yml 中 MySQL 密码（默认 123456）
4. 启动 Redis（默认 localhost:6379，无密码）
5. 启动项目，端口 8080
6. 接口统一前缀：/api
